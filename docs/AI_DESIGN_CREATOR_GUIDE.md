# AI Design Creator - User Guide

## Overview
The AI Design Creator uses Google's Gemini AI to help you create custom social media ads and posters based on your brand context and creative requirements.

## Features

### 1. **Context Setup**
Set your brand context to get personalized design recommendations:
- **Logo Upload**: Add your business logo
- **Reference Images**: Upload design inspirations or previous ads
- **Business Details**:
  - Business Name
  - Industry
  - Target Audience
  - Brand Colors (hex codes)

### 2. **Image Analysis**
- Upload reference images and tap "Analyze"
- AI will extract:
  - Color palettes
  - Typography styles
  - Layout patterns
  - Design elements
  - Mood and style recommendations

### 3. **Chat Interface**
Describe what you want in natural language:

**Example Prompts:**
```
"Create a Diwali sale poster with 50% off, use festive colors"

"Design an Instagram story for a new product launch, 
modern and minimalist style"

"Make a Facebook ad for a restaurant, 
emphasize family-friendly atmosphere"

"Create a promotional banner for a gym membership, 
energetic and motivating"
```

### 4. **AI-Generated Design Specs**
The AI will provide:
- Layout description and composition
- Typography recommendations
- Color scheme with hex codes
- Visual elements to include
- Copy/text suggestions
- Mood and style guidelines
- Technical specifications

## How to Use

### Step 1: Navigate to AI Design Creator
1. Go to the **Designs** tab
2. Tap the **"AI Design Creator"** card at the top

### Step 2: Set Up Context (Optional but Recommended)
1. Tap the **Logo** box to upload your logo
2. Tap **References** to add inspiration images
3. Fill in business details:
   - Business Name: "My Restaurant"
   - Industry: "Food & Beverage"
   - Target Audience: "Families, 25-45 years"
   - Brand Colors: "#FF6B6B, #4ECDC4"

### Step 3: Analyze References (Optional)
1. After uploading reference images
2. Tap the **"Analyze"** button
3. AI will provide insights about the images

### Step 4: Chat with AI
1. Type your design request in the input field
2. Tap send
3. Wait for AI to generate design specifications
4. Review the recommendations
5. Continue the conversation to refine

### Step 5: Implement the Design
- Use the AI-generated specs to create your design
- Or share with a designer
- Or use as inspiration for template selection

## Tips for Best Results

### 1. **Be Specific**
❌ "Make a poster"
✅ "Create a square Instagram post for a summer sale, 
    bright colors, include 30% off text"

### 2. **Provide Context**
Include:
- Platform (Instagram, Facebook, etc.)
- Purpose (sale, announcement, awareness)
- Key message or offer
- Desired mood/style

### 3. **Use Reference Images**
- Upload 2-3 images of designs you like
- Can be competitor ads, design inspiration, or previous work
- AI will analyze and incorporate similar elements

### 4. **Specify Constraints**
Mention:
- Dimensions (square, story, banner)
- Must-include elements (logo, phone number, CTA)
- Colors to use or avoid
- Text that must appear

### 5. **Iterate**
- Start broad, then refine
- Ask follow-up questions
- Request variations

## Technical Details

### Supported Features
- ✅ Text-based design generation
- ✅ Image analysis (reference images)
- ✅ Multi-turn conversations
- ✅ Context-aware recommendations
- ✅ Brand consistency

### Limitations
- AI generates **design specifications**, not actual images
- You'll need to implement the design using:
  - Design tools (Canva, Figma, etc.)
  - Templates
  - Designer
- Maximum 500 characters per message
- Image analysis works best with 2-5 reference images

## Privacy & Data

- Your context and images are sent to Google's Gemini API
- Conversations are stored locally on your device
- No data is shared with third parties
- See Google's Gemini API privacy policy for details

## Troubleshooting

### "Generation failed"
- Check internet connection
- Verify Gemini API key is configured
- Try a simpler prompt
- Reduce number of reference images

### "Analysis failed"
- Ensure images are valid (JPEG/PNG)
- Try with fewer images
- Check file sizes aren't too large

### Slow responses
- Complex prompts take longer
- Image analysis requires more processing
- Be patient, typically 5-15 seconds
